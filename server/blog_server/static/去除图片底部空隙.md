# 去除图片底部的间隙
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<style>
    .box {
        width: 220px;
        outline: 1px solid #aaa;
    }
</style>
<body>
    <p>图片下方有一定的空隙</p>
    <div class="box">
        <img src="./testImage/jibei.jpg" alt="">
    </div>
    <!-- 内联元素渲染时会有”幽灵空白节点“ -->
    <!-- 产生间隙的原因就是幽灵空白节点，lineheight，verticalalign -->
    <p>图片块状化</p>
    <div class="box">
        <img src="./testImage/jibei.jpg" alt="" style="display: block;margin: auto;">
    </div>
    <p>容器line—height足够小</p>
    <div class="box" style="line-height: 0;">
        <img src="./testImage/jibei.jpg" alt="">
    </div>
    <p>容器font-size足够小</p>
    <div class="box" style="font-size: 0;">
        <img src="./testImage/jibei.jpg" alt="">
    </div>
    <p>图片设置其他vertical-align属性值</p>
    <div class="box">
        <img src="./testImage/jibei.jpg" alt="" style="vertical-align: bottom;">
    </div>
</body>
</html>
```
